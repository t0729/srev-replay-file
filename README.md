# リアルタイム地震ビューアーのリプレイファイル
リアルタイム地震ビューアーで使用可能なリプレイファイルです。  
ファイルサイズを抑えるため、zip形式に圧縮して公開しています。   
リプレイファイルはすべて[ここ](https://github.com/t0729/srev-replay-file/tree/main/%E3%83%AA%E3%83%97%E3%83%AC%E3%82%A4%E3%83%95%E3%82%A1%E3%82%A4%E3%83%AB)にあります。    
使用等に制限は一切ありません。
## SNS等への投稿について
YouTubeやXなどの動画・投稿で使用する場合は、  
「リアルタイム地震ビューアー（改造版）で配布されているリプレイファイルを使用しました」  
などの表記をしてもらえると嬉しいです。
## 注意事項
安定性や正確性については一切保証できないため、**利用者の自己責任**で使用してください。
